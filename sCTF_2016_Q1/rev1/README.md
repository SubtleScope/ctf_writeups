# Rev1 Write Up

## Synopsis
  Reverse Engineering Challenge, find the flag

## Background
  Find the flag, the flag is not in the standard sctf{xxxx} format. You must put it in the flag format.

## Solution
  - $ `strings rev1 | less`
  - Output:
    ....
    fffff.
    h4x0r!!!H
    []A\A]A^A_
    ....
  - Flag: sctf{h4x0r!!!}
